# WT-625 — стабильный chat rail

## Цель

Устранить сдвиг desktop-интерфейса активной комнаты, когда длинная переписка
в правом чате растягивает grid-строки video, voice и diagnostics.

## Реализовано

- В active desktop workspace чат получил независимую ограниченную высоту.
- История сообщений заполняет доступное место и прокручивается внутри chat rail.
- На ширине до `920px` чат снова имеет естественную высоту в одноколоночной
  раскладке; история остаётся ограниченной `280px` и прокручиваемой.
- E2E-проверка создаёт room, отправляет два длинных сообщения и подтверждает,
  что высота chat rail не меняется, а список сообщений становится
  прокручиваемым.

## Проверки

```bash
pnpm test:e2e
pnpm format:check
pnpm check
```

## Ограничения

Фикс относится к desktop workspace. На узком экране layout намеренно
одноколоночный: чат идёт после stage, поэтому ему не нужно удерживать высоту
соседних grid-строк.
